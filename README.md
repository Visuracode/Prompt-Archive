# AI Video Prompt Archive

Welcome! This is a collection of detailed prompts for generating images and videos. 
All prompts are ready to copy and paste.

**How to use:**
- To use a Prompt click on the desired Prompt md file
- Read any specific instructions when opening
- A table of contents is included to quick link to different prompts in the file
- Click the **Copy** button next to the prompt on the right
- Paste it into your generator

---

## How to Copy Prompts
All prompts are inside code blocks. Just click the **Copy** button that appears when you hover over them.

## How to Modify Prompts
Paste the prompt into a chat with Grok and ask it to "Modify the pasted prompt" and clearly describe what changes you want

## How to Use Skill Files
- Download the skill file
- Upload it into a Grok chat (account required)
- Ask Grok *analyzie the attached skill file and create a new skill from it called [name here]*
- Once the skill is saved start a new chat
- Click on the + and select the skill
- Add any photos that you want to incorporate, now ask grok to create a prompt clearly describing what you want
- Grok will build an elaborate prompt using the skill
